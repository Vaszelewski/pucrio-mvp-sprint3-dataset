Graduate Admission - Um sistema de predição

Este sistema tem por finalidade prover um predição de admissão ou não em universidades americanas.

---

Para obter a predição, insira abaixo os dados de acordo com os valores sugeridos em cada campo e clique em "Processar" para obter o resultado.

Após o processo, a coluna "Chance of Admit" pode imprimir duas possibilidades: 0 para Negativo (sem chance de admissão) e 1 para Positivo (tem chances de admissão)

Os dados "Chance of Admit" do dataset utilizados para treinar este sistema estavam em formato decimal com valores entre 0 e 100.

Para que fosse possível a Classificação, valores que estivessem entre 0.0 e 69.9 foram definidos como 0 (Sem chance de Admissão) e valores entre 70.0 e 100.0 foram definidos como 1 (Com chance de Admissão)
